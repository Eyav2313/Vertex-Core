Place licensed SF Pro Display font files here if you own the rights to use them.
Vertex currently detects the downloaded names below:

- `SFPRODISPLAYULTRALIGHTITALIC.OTF`
- `SFPRODISPLAYTHINITALIC.OTF`
- `SFPRODISPLAYLIGHTITALIC.OTF`
- `SFPRODISPLAYREGULAR.OTF`
- `SFPRODISPLAYMEDIUM.OTF`
- `SFPRODISPLAYBOLD.OTF`

If the files are missing, Chromium falls back to Manrope.
